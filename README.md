## 欢迎大家使用bchat！

本次，我们为大家带来了bchat的新版本，1.0.1，此版本基于上次的基础上，新增了以下功能：

1. 可更改用户名：现在所有用户均可以修改用户名，而不是直接显示IPV4地址。

![](https://pic1.imgdb.cn/item/69941c182638a1fb783eca70.png)

2. 备注功能：现在用户可以给联系人中的用户进行备注，且备注后对方无法收到通知或查看备注内容

3. 登录密码保护：如果你觉得聊天环境不安全，可以启用 “登录密码保护” ，这样在每次登录时都会需要密码，防止同一个IP不同的设备登录

![](https://pic1.imgdb.cn/item/69941bcc2638a1fb783eca6d.png)

4. 云端文件管理：这是本次的一个重要功能，它可以直接让你在 “设置” 中快速的管理发送过的文件

![](https://pic1.imgdb.cn/item/69941b6c2638a1fb783eca69.png)

5. 注销账户：如果不想用了，可以直接注销账户，删除全部个人数据

![](https://pic1.imgdb.cn/item/69941c8b2638a1fb783eca76.png)
